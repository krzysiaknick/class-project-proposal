Pogue Timer

Nick Krzysiak

This product will be a web app which contains a simple timer which can be used to count down from various invervals
of time in a simple and minimalist approach. The timer will show minutes remaining only as requested by Professor Pogue.

The application will be hosted on the web using Github Pages and will incorporate HTML, CSS, and JavaScript

  • Simple and easy to use interface
